Attendance Information for Teachers - School Managed Settings

Setting: Disable Attendance Info for Teachers for ~(schoolname)
Default setting is for the features of the Attendance Information for Teachers plugin to be enabled for all schools. Checking this box will turn the features off for only ~(schoolname).

Attendance Information for Teachers - School Managed Settings

Setting: Only show school abbreviations if the class does not meet at the current school
Default setting shows the abbreviation of the school where each class in the student's schedule is taught. Checking this box will remove the abbreviation for classes taught at the teacher's current school and the abbreviation will appear only for classes taught at other schools.

Attendance Information for Teachers - School Managed Settings

Setting: Show only classes scheduled at ~(schoolname) in the student's schedule
Default setting shows the student's entire schedule, including classes taken at other schools (dual/remote enrollments) on the same PowerSchool server. Checking this box will remove those classes from the student's schedule listing.

Attendance Information for Teachers - School Managed Settings

Setting: Include courses/sections that have been set to be excluded from ADA calculations
Default setting shows only classes that count toward ADA (average daily attendance) calculations. Checking this box will show all current course enrollments for the student, regardless of whether attendance taken for the class counts towards ADA calculations.

Attendance Information for Teachers - School Managed Settings

Setting: Show the red chair for all attendance codes, not just absences
Default setting will display the red chair when the student has at least one attendance record, meeting or daily, with a presence status of Absent. The chair will not display when all attendance codes on record for the day and for the student have a presence status of Present (e.g. tardies, excused codes such as field trips or athletics, or any code indicating the student may or may not have been in class but was still counted as present.) Checking this box will display the red chair when the student has any attendance records, regardless of presence status, that are not the default Present code.
